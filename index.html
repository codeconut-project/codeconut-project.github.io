Code Coverage for C and C++ code

Codeconut is a open source and free code coverage tool for C and C++ code. It provides statement, decision and MC/DC coverage and adheres to guidelines made by standards in the automotive and aviation sector.

Learn more

Why use Codeconut?

Codeconut provides coverage metrics that correlate to the logic you write in your code instead of instrumenting the code on assembler level.
This enables the user to create tests with more focus on the logic that is written in code.
